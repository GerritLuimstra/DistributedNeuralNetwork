What is our project?

The project is based on Neural Networks. The initial idea is to be able to increase the spead in which a Neural Network learns through an inexpensive method. One of our end goals for the project is to be able to have a fully functioning Neural Network which is capable of analyzing lung x-rays and determine with a high percentage of accuracy if the the patient has pneumonia or not.
